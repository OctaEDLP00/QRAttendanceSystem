# QR Attendance System

This project is a QR code-based attendance system that allows users to generate and scan QR codes for attendance tracking. It is built using NestJS for the backend, Astro for the frontend and React Native (Expo) for the mobile app.

## Features

- Generate QR codes for attendance
- Scan QR codes to mark attendance
- View attendance records
- User authentication and authorization

## Technologies Used

- NestJS: A progressive Node.js framework for building efficient and scalable server-side applications.
- Astro: A modern frontend framework for building fast and optimized web applications.
- React Native (Expo): A framework for building native mobile applications using React.

## Installation

1. Clone the repository:

```bash
git clone
cd qr-attendance-system
```
2. Install dependencies:

```bash
pnpm i
```

3. Start the development server:

```bash
pnpm start
```
4. Start the mobile app:

```bash
cd mobile
expo start
```

## Usage

- Access the web application at `http://localhost:3000`.
- Use the mobile app to scan QR codes for attendance.
